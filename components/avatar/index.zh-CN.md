## 参数

| size        |     大小 |          `large` `normal` `small`          | `normal` |      |

| customStyle |    style |     <font color=#c41d7f>object</font>      |    -     |
| onClick     | 点击事件 | <font color=#c41d7f>(event) => void</font> |    -     |
