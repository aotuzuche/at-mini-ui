## AT-Console-Components

基于 [Antd](https://github.com/ant-design/ant-design)，对常用**业务**组件封装扩展，提高开发效率。

- [x] **100% Typescript**
- [x] 兼容原生组件

## 使用

### 安装

```bash
$ yarn add at-console-components
```

### 组件

```js
import { AsyncButton } from 'at-console-components'

function Demo(params) {
  return <AsyncButton />
}
```
